## [–ó–Ω–∞—á–µ–Ω–∏–µ –º–Ω–æ–≥–æ—á–ª–µ–Ω–∞ üå∂Ô∏è](https://stepik.org/lesson/512854/step/16?unit=505068)
### –ó–∞–¥–∞–Ω–∏–µ
–ú–Ω–æ–≥–æ—á–ª–µ–Ω–æ–º¬†—Å—Ç–µ–ø–µ–Ω–∏ nn –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è –≤—ã—Ä–∞–∂–µ–Ω–∏–µ –≤–∏–¥–∞¬†a_nx^n + a_{n-1}x^{n-1}+ ... + a_2x^2+a_1x+a_0

**–§–æ—Ä–º–∞—Ç –≤—Ö–æ–¥–Ω—ã—Ö –¥–∞–Ω–Ω—ã—Ö**  
–ù–∞ –≤—Ö–æ–¥ –ø—Ä–æ–≥—Ä–∞–º–º–µ –Ω–∞ –ø–µ—Ä–≤–æ–π —Å—Ç—Ä–æ–∫–µ –ø–æ–¥–∞—é—Ç—Å—è –∫–æ—ç—Ñ—Ñ–∏—Ü–∏–µ–Ω—Ç—ã –º–Ω–æ–≥–æ—á–ª–µ–Ω–∞ (—Ü–µ–ª—ã–µ —á–∏—Å–ª–∞),¬†—Ä–∞–∑–¥–µ–ª–µ–Ω–Ω—ã–µ¬†—Å–∏–º–≤–æ–ª–æ–º –ø—Ä–æ–±–µ–ª–∞¬†–∏ —Ü–µ–ª–æ–µ —á–∏—Å–ª–æ xx –Ω–∞ –≤—Ç–æ—Ä–æ–π —Å—Ç—Ä–æ–∫–µ.

**–§–æ—Ä–º–∞—Ç –≤—ã—Ö–æ–¥–Ω—ã—Ö –¥–∞–Ω–Ω—ã—Ö**  
–ü—Ä–æ–≥—Ä–∞–º–º–∞ –¥–æ–ª–∂–Ω–∞ –≤—ã–≤–µ—Å—Ç–∏ –æ–¥–Ω–æ —á–∏—Å–ª–æ¬†‚Äî¬†–∑–Ω–∞—á–µ–Ω–∏–µ —É–∫–∞–∑–∞–Ω–Ω–æ–≥–æ¬†–º–Ω–æ–≥–æ—á–ª–µ–Ω–∞ –ø—Ä–∏¬†–∑–∞–¥–∞–Ω–Ω–æ–º –∑–Ω–∞—á–µ–Ω–∏–∏ xx.

### –†–µ—à–µ–Ω–∏–µ

from functools import reduce
from operator import add


def evaluate(coefficients, x):
    return reduce(add, map(lambda a, b: a * x ** b, coefficients, range(len(coefficients) - 1, -1, -1)), 0)


print(evaluate([int(i) for i in input().split()], int(input())))
